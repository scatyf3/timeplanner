# 回填示例（timeplanner → Obsidian 日记）

跨日 hook（[rollover.py](../timeplanner/core/rollover.py)）在第二天首次运行任意 CLI 命令时，
把昨天的本地 timeline 写回当天的 Obsidian 日记，然后从本地缓存删除。

下面是**回填后**一篇日记的完整样子：你原有的内容原封不动，只在末尾多出一个
`## ⏱ 时间线（timeplanner 回填）` 小节，用共享时间轴把 ①Plan / ②Actual / ③AW观测 三条线**并列**起来。

在 Obsidian 预览里，那个表格会渲染成三列对照，一眼看出计划和实际的偏差。上例中：

- **15:30** 那行：Plan 本该继续 `拆 PR`，Actual 却是 `摸鱼刷知乎`，AW 也显示在机 —— 计划外的岔开一目了然。
- **20:00** 的 `健身房`：Plan 里排了，Actual / AW 都空 —— 当天没做。

## 格式规则

- **小节标题**固定为 `## ⏱ 时间线（timeplanner 回填）`，作为「机器写入区」的锚点；重复运行只替换这一节，不动其它内容。
- **时间轴**按 30 分钟一格（与 `timeplanner summary` 的终端渲染同粒度）。
- **色块** `█` 表示这一格被某个事件占用；**只在事件起始格显示标题**，其余格只留 `█`，于是能看出块的时长。
- **③ AW观测** 列只在当天有 focus block 时出现；AW 没开也不影响回填。
- **全空的行会被折叠**（长的一天表格也不会爆）；显示出来的每一行都带时刻。
- 标题里的 `|` 会转义，过长的 summary 截断到 16 字。
